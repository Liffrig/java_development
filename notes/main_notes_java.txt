General:
	JDK - Java Development Kit 
		- compile and run java code

Code Stucture and conventions:
	- clas with the same name as file -> inside public static void main()
	- use PascalCase for naming convention
	- use camelCase for variables
	- Java is case sensitive -> int value != int Value
	- dispose object to avoid resource leak
	- Scanner uses white space as delimiter
	- 

	- int + String compiles and gives a string 
	- char + String compiles and gives a string 
	- == or != for int BUT .equals or .!equals for string 
	- || or 
	- && and
